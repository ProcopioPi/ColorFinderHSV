

ARGOTH.sln : Es la solución utilizada desde C# 
	
SNIPS\MON\ColorFinder.cs :: (ruta al archivo ColorFinder)
	El archivo ColorFinder.cs es la clase que lleva a cabo 
	el análisis de la imágen pixel a pixel convirtiendo de RGB a HSV
	para cambiar los valores encontrados entre un mínimo y un máximo de 
	los valores del disco.
	
ARGForm :: archivo CS de la interfáz gráfica.
	 obtiene el directorio de una imagen seleccionada para empezar a buscar
	 los colores deseados.(BTN_OPEN_IMAGE_Click)
	 También se puede después mandar a procesar iterativamente
	 todos los archivos en la carpeta con los valores seleccionados.(ProcessFolder)
